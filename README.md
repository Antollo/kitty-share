# Kitty share

Social media application using React for frontend and Node.js for backend (Express-based server and MongoDB database).

Features:

- Login with Facebook, Google and GitHub.
- Make posts.
- Like and dislike posts.
- Comment posts.
- Posts rendered using Markdown.

Live demo: [kitty-share.herokuapp.com/](https://kitty-share.herokuapp.com/)

### Install, build and run

```
cd client
npm install
npm run build
cd ..
npm install
npm start
```

### Debug frontend

```
cd client
npm run start
```
